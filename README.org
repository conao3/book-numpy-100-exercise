* Overview

My solutions to the [[https://github.com/rougier/numpy-100][100 NumPy Exercises]], a collection of exercises ranging from basic to advanced NumPy operations. The exercises cover array creation, indexing, mathematical operations, broadcasting, and more.

* Contents

- =solution.org= - All 100 exercise solutions with explanations and output

* Requirements

- Python 3.x
- NumPy

* Getting Started

Clone the repository and explore the solutions:

#+begin_src shell
git clone https://github.com/conao3/book-numpy-100-exercise.git
cd book-numpy-100-exercise
#+end_src

The solutions are written in Org mode with embedded Python code blocks. You can:

- View them directly on GitHub
- Open in Emacs for interactive execution with Org Babel
- Export to other formats using Org mode's export features
